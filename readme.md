# Skincare Ingredient Scout

An AI-powered web application that helps users identify potential conflicts and interactions between skincare ingredients, ensuring safer and more effective skincare routines.

## ğŸ¯ About This Project

Skincare Ingredient Scout was created by a team of students who experienced skin problems from mixing incompatible skincare products. Many teenagers and skincare enthusiasts face similar issues - using products together that cause irritation, redness, or allergic reactions. This project aims to be a helpful tool that reminds people, especially teenagers, which products can and cannot be used together.

## ğŸš€ How to Use

1. Open `index.html` in your web browser
2. Enter your skincare ingredients (separated by commas, semicolons, or new lines)
3. Click "Analyze" to get instant results
4. Review the analysis to see if your ingredient combination is safe or has potential conflicts

The application works completely offline and uses intelligent analysis to detect common ingredient conflicts like:
- Retinol + Acids (AHA/BHA)
- Vitamin C + Niacinamide
- Multiple active ingredients that may cause irritation
